Specialties

Web Development

Developing Software is an amazing feeling and web development is a very broad topic, so I will be clear on what it is I do. Solving problems with logic, adding value to your company, by writing programs that automate services and allow your data to be accessible when needed while also creating data that is secure. UI/UX

Specializing in the visual aesthetic of an application including the transitions, interface animations and any sort of visual element is art to me. Adding the simplicity of how a user interacts with that art is a passion I fullfil on with each assignment. Merging both the user interface and the user experience. Backend Services
